发表于 2016-05-19   |  

简介

在本章中，主要介绍了vue项目基于leancloud后台实现用户的注册登录功能，同时也可以实现注册时的短信验证码、第三方注册等功能。

Leancloud

Leancloud入门

用户可以首先申请一个leancloud免费帐号，然后新建一个项目，并得到相应的AppID、AppKey，本项目是基于leancloud的javascript平台进行开发的，使用的是leancloud数据存储开发功能，相应的API接口文档

安装Leancloud SDK

项目安装Leancloud模块，如下

npm install avoscloud-sdk –save-dev

相应的vue使用Leancloud如下：

import AV from ‘avoscloud-sdk’
AV.initialize(‘AppID‘, ‘AppKey‘)

简介
