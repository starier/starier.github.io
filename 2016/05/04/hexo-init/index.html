hexo框架的使用总结

发表于 2016-05-04   |  

开发模式

启动查看效果

hexo server

打开浏览器http://localhost:4000，即可看到效果页面

新建一篇文章

例如最近写的一篇文章Vue+webpack基础教程（一），新建这篇文章的命令如下

hexo new -p vue-webpack-init Vue+webpack基础教程（一）

文章创建成功后，编写相应的md文件即可

上线部署

文章编写完成后，本地测试通过后，执行hexo generate命令，编译完成后执行hexo deploy即可完成文章的发布
