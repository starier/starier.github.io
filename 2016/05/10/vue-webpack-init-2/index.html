发表于 2016-05-10   |  

简介

在上一教程中，主要介绍了Vue项目的页面构建，即组件设计，在本教程中，主要讲解了Vue项目的路由设计及配置开发，包括路由基础配置、子路由管理等内容。

前提条件

在Vue项目中，尤其是单页应用，路由功能主要由vue-router完成，官方中文API，项目使用vue路由之前，需要安装相应插件，执行以下命令

npm install –save-dev vue-router

同时，项目中也用到了vue-strap，用来实现vue中的bootstrap效果，还有leancloud，用户实现系统的后台接口，如用户的注册、登录等功能
